Entrepreneurs' Trivia


Description

This event aims to promote awareness and persistence in various fields of work by laying a small quiz session on the theme 'innovation'.


Details

Participant who answers the fastest will be given the point for that question Participants and best-performers will get certificates.


Topic

1. Entrepreneurship and entrepreneurs.
2. Famous start-ups (Indian and global).
3. Start-up related
4. Engineering trivia.


Guidelines

1. This event is a short, mixed quiz for all the participants. It will have 13 questions in total.
2. 25 seconds x 5 questions x 4 rounds (elimination after each round). plus, facts in between questions and rounds.
3. ' Kahoot! ' game-based learning platform will be used to organize the quiz; it can be easily accessed via a web browser or the Kahoot app.
4. Winners will get an e-certificate
